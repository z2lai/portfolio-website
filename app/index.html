My Work

React Multi-step Form Library

I built a React library for users to easily create interactive multi-step forms using declarative code. The library handles form state management using the Context API, custom hooks and custom input components. All inputs are displayed one at a time, with input validation occurring when the end user clicks the button to proceed to the next input.

Github Website

Highlights

Declarative Configuration

My focus for this project was to keep the form setup code as simple as possible so I implemented the input components using custom hooks and compound components pattern to allow the user to declaratively configure the inputs.

Optimized Form State Management

Form state management is a common problem that I wanted to solve using concepts such as state colocation and logically separated contexts, I've minimized the number of unnecessary renders.

Optimized Page Transitions

Interactivity in applications is one of the first things that drew me to become a web developer so I made sure to include that in my UI components by implementing the expansion/collapsing of the form body as the size of the input component changes. In order to keep this animation smooth, I made sure to avoid reflow by creating a scale animation for the form body and an inverse scale animation for the form contents using an easing function and a custom hook.

HTML5 Form Validation

A key factor of good usability is conforming to user's expectations, so I made sure that input validation aligned with the HTML5 form validation standard by using the Constraint Validation API. Users are also able to specify custom error messages or a custom validation function altogether.
